gsTiles.optTile module¶

Filename: optTile.py Use: Optimzes Tiles for gsTiles

determines the best format to store the raster. Outputs are: singleton, plte, full raster.

Usage: > tileType,strTile = optTile(png)

class gsTiles.optTile.PLTE(intRaster, regions, n, width=None)[source]¶

Palletizes an intRaster and packs it into a base64 encoded string. 9 byte header: byte0 = ByteOrder byte1:4 = in ”!” network byte order len of IDAT byte4:8 = in ”!” network byte order len of PLTE byte9:n = ready to go IDAT (in network (big) byteorder) byten: = cPLTE

Raises IOError is raster has too many regions (>256)
